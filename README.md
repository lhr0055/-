# 김치 웹 사이트

이 프로젝트는 김치에 관한 다양한 솔루션을 제공하는 웹 사이트 입니다. 이용자들은 다양한 김치를 검색하고, 확인하고, 관리할 수 있습니다. Ennjoy KIMCHI!!! 

## 김.치.조.아

## 기능

- 다양한 김치 레시피 검색
- 각 레시피의 재료와 조리법 상세 보기
- 사용자 리뷰 및 평점 기능
- 반응형 디자인으로 모든 디바이스에서 최적화된 화면 제공
